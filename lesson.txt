*when compiling a c program, the user can define a value.
for example : 
    $ gcc -D MYDEFINE=value -D MYDEFINE2=value2 -o outputfile sourcefile.c

    sourcefile.c might have a " #ifndef MYDEFINE
                                    #define MYDEFINE init
                                #endif
                                #ifndef MYDEFINE2
                                    #define MYDEFINE2 init_else
                                #endif " before the code or any function using the defined values.



*when compiling a c program, the user can optimize the output executable file
    $ gcc -O0 -o outputfile sourcefile.c // no optimisation
    $ gcc -O1 -o outputfile sourcefile.c // little optimization
    $ gcc -O2 -o outputfile sourcefile.c // big optimization
    $ gcc -O3 -o outputfile sourcefile.c // the best optimization


*the user can know how many time an executable took by adding "time" before the executable name
    $ time ./outputfile
